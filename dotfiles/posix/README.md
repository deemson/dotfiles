# posix

Configuration shared across macOS and Linux.

- **dotbin** — personal shell scripts and utilities, available on `$PATH`
- **mise** — tool version manager; installs and switches runtimes (Node, Go, Bun, Lua, etc.)
- **neovim** — text editor; configured with LSP, completion, debugging, and a plugin ecosystem
- **starship** — shell prompt; shows context like git branch, language versions, and exit codes
- **yamlfmt** — YAML formatter
- **zsh** — shell configuration and plugins
